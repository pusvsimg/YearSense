# 🧧 YearSense — 年味指数检测器

> 14 道题，测出你今年的年味浓度

🌐 **在线体验：** https://yearsense.ndjp.net

## 功能特色

- 🎴 **翻页式答题** — 每次一题，选完自动滑到下一题，支持左右滑动手势和键盘方向键
- 🧮 **智能评分算法** — 14 项基础分值 + 9 种组合加成（如"完美除夕夜""红包大赢家""阖家团圆"）
- 🏆 **8 档年味等级** — 从"年味稀薄 ☕"到"年味天花板 🏆"
- 🎆 **动态粒子背景** — 红金双色粒子飘落，高分时触发烟花爆炸
- 📊 **SVG 环形动画** — 分数渐变环 + 数字跳动展示
- 📱 **全屏沉浸体验** — 欢迎页 → 答题页 → 报告页三屏切换，无需滚动

## 交互流程

```
欢迎页 ──点击开始──▶ 答题页（14题翻页）──全部作答──▶ 年味报告页
                         ▲                                │
                         └────────── 重新测评 ◄────────────┘
```

- 答题页顶部：圆点导航（红色=已答 / 金色发光=当前）+ 进度条
- 支持前后箭头、圆点跳转、手势滑动、键盘左右键

## 技术实现

- **纯前端** — HTML + CSS + Vanilla JS，无任何依赖
- **三屏切换** — CSS transform + opacity 过渡动画
- **翻页轮播** — Flexbox 横向排列 + translateX 滑动，支持 Touch Events 手势
- **粒子系统** — Canvas 2D 绘制，含常驻飘落粒子和烟花爆炸粒子
- **SVG 环形图** — stroke-dashoffset 动画 + linearGradient 渐变描边
- **响应式** — 适配桌面端和移动端，全屏高度布局
